## Design

- **Colours:** Warm beige `#fbf9f5` background, teal `#2eb5b1` / `#006a67` accents
- **Fonts:** Noto Serif (headings), Work Sans (body) — both via Google Fonts
- **Layout:** 1120px max-width, 12-column grid, responsive
